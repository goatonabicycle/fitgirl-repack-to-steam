# Privacy Policy for FitGirl Repack to Steam

Last updated: December 21, 2024

## Overview

FitGirl Repack to Steam is a browser extension that displays Steam game information on FitGirl Repacks pages. This extension is designed with privacy in mind and collects no personal data.

## Data Collection

**This extension does not collect, store, or transmit any personal information.**

## Local Storage

The extension stores the following data locally in your browser:

- **Game cache**: Search results from Steam are cached locally for 5 days to reduce API requests and improve performance
- **Display preferences**: Your settings (which information to show, whether to open links in Steam client, etc.)

This data never leaves your device and is not accessible to anyone except you.

## External Requests

This extension makes requests to Steam's public API (`store.steampowered.com`) to fetch game information. These requests include:

- The game name extracted from the page you're viewing
- Standard HTTP headers sent by your browser

**No cookies, user identifiers, or personal data are included in these requests.**

## Third Parties

- No analytics or tracking services are used
- No data is sold or shared with third parties
- No advertising networks are integrated

## Permissions

This extension requires the following permissions:

- **Host permission for steampowered.com**: Required to fetch game information from Steam's API
- **Storage**: Required to save your preferences and cache game data locally

## Open Source

This extension is open source. You can review the complete source code in this repository.

## Changes to This Policy

Any changes to this privacy policy will be reflected in this document with an updated date.
